면역학 STUDY (5) - Primary Response vs. Secondary Response

Booster Effect

Primary response 이후에 강한 secondary response를 주는 것을 booster effect라고 부른다.

booster effect를 최대한 강하게 만들기 위해서는 다시 immunize 시키는 것을 Ab의 농도가 거의 0에 가까울 때 해야하는데 만약 이 시점보다 너무 빨리 immunize하게 되면 주입한 Ag가 primary response에 의해 줄어들어서 두번째 immunization 반응을 크게 유도할 수 없다. 반면 너무 늦게 하게 되면 memory cell의 population이 크게 줄어서 secondary response가 크게 일어나기 힘들다.

secondary response는 primary response에 비해

  1. lag phase가 짧게
  2. 더 빠르게
  3. 짧은 시간 동안
  4. 더 강한 affinity를 가진 antibody들의 작용이 일어난다.

affinity maturation로 secondary response는 primary에서 IgM을 생성하는 것 과 달리 class switch가 일어나서 affinity가 높은 IgG Abs가 존재한다. 즉 titer가 높은 Abs가 존재한다고 할 수 있다.

Note B cell은 Hapten을 인식하고 T cell은 Carrier를 인식한다?
